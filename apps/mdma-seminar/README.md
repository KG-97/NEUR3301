# MDMA Seminar source track

The deployed static seminar studio lives at `docs/seminar/`.

Canonical source material in this repository:

- `docs/seminar/index.html`
- `docs/resources/MDMA_Seminar_Full_Script_15min.md`

The July 2026 hardened version corrects several common overclaims:

- mouse critical-period evidence is not presented as demonstrated human PTSD mediation;
- extracellular-matrix remodelling is not upgraded into proven human perineuronal-net remodelling;
- plasma oxytocin is not treated as proof of a single empathy mechanism;
- Phase 3 results are described as a treatment-package signal;
- FDA advisory votes and the regulatory critique are stated precisely;
- zero adverse-event reports are not interpreted as zero adverse events;
- supplemental dosing is described as increasing exposure/duration under nonlinear kinetics rather than automatically increasing peak effects.

The public build intentionally has no API-key or cloud-auth dependency.
