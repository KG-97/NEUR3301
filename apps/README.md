# NEUR3301 apps

Place complete source applications in separate folders beneath this directory.

```text
apps/
├── synapse/
├── mdma-seminar-studio/
└── future-tool/
```

Each app folder should include its own README with:

- purpose and assessment target
- local setup and build commands
- deployment URL
- scientific sources or evidence notes
- known limitations

The currently deployed Synapse app is registered in `../docs/apps.json`. Its complete source bundle still needs to be imported here.
