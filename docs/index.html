Default study loop

Compress → retrieve → interrogate → repair

For each lecture: reduce the mechanism to a causal chain, reproduce it closed-book, predict an intervention, then encode only the errors.

  1. 01
    Compress

    Write the lecture’s causal chain in six arrows or fewer.

  2. 02
    Retrieve

    Redraw and explain it without notes.

  3. 03
    Interrogate

    Predict a blocker, knockout, lesion or dose effect.

  4. 04
    Repair

    Log the exact failure and retest the corrected mechanism.
